# Password-Checker
A security tool that evaluates the strength, complexity, and security of a password, often estimating how long it would take to crack. Evaluates if a password is weak, moderate, or strong based on entropy (randomness) and complexity. Estimates the time it would take hackers to guess the password using brute-force tools.
